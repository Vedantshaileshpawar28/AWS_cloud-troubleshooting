# Incident 02: Application Inaccessible Due to Security Group Misconfiguration

## Problem Description

A web application hosted on an EC2 instance was reported as **not accessible from the internet**.  

- Users reported that the website could not be reached (connection timed out).  
- Instance is running, and application service (Nginx) appears to be running.  
- No recent code deployments or configuration changes were done on the application itself.
